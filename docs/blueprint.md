# **App Name**: RedeInovaSocial

## Core Features:

- Authentication: Secure user authentication using Firebase Auth (email/password).
- News Feed: Display news from Firestore collection `news` on the dashboard.
- Task Management: List pending tasks from Firestore collection `tasks`, filtered by user ID.
- Team Flowchart: Visualize team structure with React Flow, with dynamic nodes and member details.
- Photo Gallery: Display photo grid from Firebase Storage; allow uploading by team members.
- Role Based Authorization: Determine roles like 'coordinator', 'researcher', or 'fellow' in Firestore.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to reflect agriculture and sustainability.
- Background color: Desaturated beige (#F5F5DC) to provide a soft backdrop.
- Accent color: Burnt orange (#CC6633) for highlights and interactive elements.
- Body and headline font: 'PT Sans' for a modern, approachable feel.
- Use clean, line-based icons related to food, technology, and collaboration.
- Clean and simple layout, focusing on readability and visual clarity. Use Tailwind CSS grid for responsiveness.
